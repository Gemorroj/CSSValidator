# Port PEAR package [CSSValidator](http://pear.php.net/package/Services_W3C_CSSValidator)


### Requirements:

- PHP >= 5.3

### Installation:

- Add to composer.json:

```json
{
    "require": {
        "gemorroj/cssvalidator": "dev-master"
    }
}
```
- Install project:

```bash
$ php composer.phar update gemorroj/cssvalidator
```
